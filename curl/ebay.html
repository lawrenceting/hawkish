Canon IP7260 Photo Inkjet Printer

+ Duplexer + Wireless + CD/DVD Printing + AirPrint

BRAND NEW in Retail Box+Manufacturer Warranty+Tax Invoice Included !

The PIXMA iP7260 features 9600 x 2400 dpi resolution for premium quality document and photo prints. Featuring creative and convenient printing features such as disc printing, auto double sided printing and 2 paper trays. Wi-Fi enables multiple users to connect from virtually anywhere in the home or office. It also enables mobile printing from both Apple and Android devices.

Specification:

Ink Droplet Size
1pl/2pl/5pl (C/M), 5pl (Dye BK/Y), 12pl (PgBK)

 

Printing Resolution(1)
9600 x 2400 dpi

 

Print Speed (2)
B&W: 15.0 ipm; Colour: 10.0 ipm; 4”X6” photo: Approx. 21 seconds

 

Ink Tanks
PGi650BK, CLi651BK, CLi651C, CLi651M, CLi651Y (optional XL ink tanks also available)

 

Ink Yield (Borderless, 4”X6” photo)
PGI-650BK / XL (2425 / 5616), CLI-651C / XL (121 / 267), CLI-651M / XL (132 / 313), CLI-651Y / XL (130 / 274), CLI-651BK / XL (376 / 1130)

 

Ink Yield (Plain paper, A4)
PGI-650BK / XL (300 / 500), CLI-651C / XL (332 / 695), CLI-651M / XL (319 / 680), CLI-651Y / XL (344 / 695), CLI-651BK / XL (1795 / 5530)

 

Borderless Printing (3)
4"x 6", 5"x 7", 8"x10", Letter, A4

 

Plain Paper Capacity
125 sheets

 

Supported Paper Sizes
A4, Letter, Legal, A5, B5, Envelopes (DL, COM10), 4"x6", 5"x7", 8"x10"

 

Supported Paper Weights
Plain Paper: 64-105g/m². Canon specialty paper (max paper weight): approx 300g/m² (PT-101)

 

Protocol
TCP/IP

 

Wireless LAN Network Type
IEEE802.11n, IEEE802.11g, IEEE802.11b

 

Wireless LAN Data Rate
IEEE802.11n: Max.150Mbps; IEEE802.11g: Max. 54 Mbps; IEEE802.11b: Max. 11 Mbps

 

Wireless LAN Range
Indoor 50m (depends on the transmission speed and conditions)

 

Wireless LAN Security
WEP64/128bit, WPA-PSK(TKIP/AES), WPA2-PSK(TKIP/AES)

 

Mobile Phone/Tablet Printing
Print from any iOS or Android mobile device through Canon EPP app, available to download at App Store (Apple) and Play Store (Android). Apple AirPrint also supported.

 

Camera/Memory Card Printing
WiFi PictBridge Printing with compatible Canon Cameras. Memory Card Slots – N/A

 

PC Interface
Hi-Speed USB
